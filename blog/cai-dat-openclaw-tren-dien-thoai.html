5 bư𞶼 setup OpenClaw — mỗi bư𞶼 có thể block người mới

  1. Cài Node.js 24+ — Tích hợp NodeSource vào Linux, check vếi node -v. Sai version → có thể mất 30 phút debug.
  2. Cấu hໃnh API key — Tạo tài khoản Anthropic, mua credit, copy key đúng format sk-ant-api03-..., paste đúng chỗ trong openclaw.json. Sai chỗ → lỗi 401 không rõ lý do.
  3. Setup Telegram Bot — Tạo bot qua BotFather, lấy token, pair với OpenClaw. Sai allowlist hoặc requireMention → bot im lặng.
  4. Deploy lên VPS — Mua server, cài Ubuntu, SSH vào, clone repo, chạy gateway. Tắt terminal → bot chết.
  5. Cấu hໃnh daemon + uptime — Dùng pm2 hoặc systemd để gateway chạy 24/7. Sai config → bot chạy rồi tự tắt sau vài tiếng.
